Simple Node.js API

This API allows CRUD operations on Movies, Series, and Songs.

Available Endpoints:

  • GET /movies - Fetch all movies
  • GET /series - Fetch all series
  • GET /songs - Fetch all songs
  • POST /movies | /series | /songs - Add a new item
  • PUT /movies/:id | /series/:id | /songs/:id - Update an item
  • DELETE /movies/:id | /series/:id | /songs/:id - Delete an item
